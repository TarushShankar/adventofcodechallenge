# Advent of Code 2024 — Day 2

Solution to [Advent of Code 2024, Day 2](https://adventofcode.com/2024/day/2).


## Run

```bash
python3 solution.py
```

Reads `data/example.txt` and prints, e.g.:

```
2 of 6 reports are safe!
```

## Files

- `solution.py` — the solution
- `data/example.txt` — sample input from the puzzle
